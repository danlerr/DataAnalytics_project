Il dataset (https://archive.ics.uci.edu/dataset/572/taiwanese+bankruptcy+prediction) contiene dati finanziari dal 1999 al 2009 provenienti dal Taiwan Economic Journal. 

Il dataset è composto da 95 caratteristiche finanziarie (indicatori) e 6819 aziende. Ogni riga rappresenta quindi un insieme di dati di bilancio di una specifica 
azienda in uno specifico anno tra il 1999 e il 2009. Il target (Bankrupt?) del dataset ci dice se quell'azienda è fallita oppure no (1:fallita, 0:non fallita).
